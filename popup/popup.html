About Media Extractor Pro
Media Extractor Pro
Version 1.0.0  ·  Manifest V3

A powerful browser extension that instantly detects and extracts all images and videos from any webpage. Works 100% locally — no data is ever uploaded.

✨ Features
  • 🖼️ Extract JPG, PNG, GIF, WEBP, SVG, AVIF, BMP, ICO
  • 🎥 Detect MP4, WEBM, OGV + YouTube & Vimeo embeds
  • 🔍 Filter by format, dimension size & keyword search
  • ⬇️ Bulk select & download with smart queue
  • 📋 Copy any media URL with one click
  • 🔒 100% local — no account, no tracking, no uploads
